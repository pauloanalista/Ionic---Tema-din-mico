# Ionic-com-tema-dinamico
Exemplo de como definir um tema dinâmico pra projetos em Ionic.

# Criar o projeto


```sh
ionic start ThemeIonic blank
cd ThemeIonic
ionic g provider settings
```

Crie um provider onde iremos gerenciar nossas configurações
```sh
ionic g provider settings
```
